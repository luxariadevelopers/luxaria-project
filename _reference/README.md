# Archived Express monorepo (reference only)

This folder contains the previous npm/Express Luxaria ERP implementation.

Use it as a **domain and behavior reference** while rebuilding on NestJS + pnpm.

Do **not** import these packages from `apps/*` or `packages/*` in the new monorepo.

Local secrets that were present at archive time (for example `apps/api/.env`) may still exist under this tree; they remain gitignored.
