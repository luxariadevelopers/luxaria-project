# Create an Employee

1. Sign in as an administrator with `employee.create`, `user.create`, `project_access.assign`, and `site_access.assign`.
2. Open **Administration → Employees**.
3. Click **Create Employee**.
4. Enter personal details (name, email, mobile, optional employee code).
5. Select department and the designation they are appointed for (e.g. Site Engineer).
6. Optionally select 1–2 reporting officers (primary + approval rule: any one / all).
7. Enable **Create login**, set a temporary password (min 8 characters).
8. On **Access**, tick the web/mobile modules they should see (untick to hide dashboards, DPR, stock, petty cash, etc.).
9. Select the assigned **Project** and **Site** (create a site if none exist).
10. Review and submit.

After save:

- Employee appears in the list  
- Login user is linked  
- Project and site assignments are active  
- Access summary is available on the employee detail / access page  

The employee can sign in and will only see assigned projects/sites and the modules you left enabled.
